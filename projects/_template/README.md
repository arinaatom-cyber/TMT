# Project results folder

Create one folder per **Project ID** (e.g. `PXD012345/`) with:

- `README.md` — short description, PMID, organ(s)
- `protein_table.csv` or `*.tsv` — protein list (columns: Gene, Protein names, UniProt — any subset)
- Name should match **Result Files** in the Google Sheet when possible
- `sample_metadata.csv` — sample annotations (optional)

The atlas links here: `https://github.com/arinaatom-cyber/TMT/tree/main/projects/PXD012345`

Rename this `_template` folder when adding a real project.
