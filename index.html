About the Atlas

Interactive map of TMT multiplex proteomics projects grouped by organ. Data is loaded from Google Sheets (PRIDE, CPTAC, PDC, ProteomeXchange).

Methods

  • One project ID = one unique project (PXD preferred when dual-listed).
  • Multi-organ rows assign each organ; ≥3 organs also tagged Multiple Organs.
  • Pan-organ atlases (≥8 organs) show a PAN-ORGAN badge.
  • Disease labels are grouped (e.g. NSCLC → Lung cancer).
